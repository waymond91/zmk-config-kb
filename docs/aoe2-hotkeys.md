## Piantor Pro BT - AoE2 Layer Quick Reference

Enter AoE2 mode: **NAV layer (hold mo3) + E**
Exit AoE2 mode: **Right-hand `to 0` key**

### AOE2 Base Layer (Layer 4) - Left Hand Only

```
TAP action (grid command)  /  HOLD action (Go-To building)

| ,/TAB |  Q   | W/MIL| E/STB| R/SIE| T/KRE|
| CTRL  | A/ARC| S/BLS| D/DOC| F/UNI| G/MIN|
| SHIFT | Z/LUM| X/MON| C/MKT| V/CAS| B/BAR|
              | GRP  | SPACE| ./H  |
```

#### Key Behaviors:

| Key | Tap | Hold (200ms) | Go-To Target |
|-----|-----|-------------|--------------|
| TAB pos | `,` (idle military) | TAB | - |
| W | W | Ctrl+W | Mill |
| E | E | Ctrl+L | Stable |
| R | R | Ctrl+K | Siege Workshop |
| T | T | Ctrl+T | Krepost |
| A | A | Ctrl+A | Archery Range |
| S | S | Ctrl+S | Blacksmith |
| D | D | Ctrl+D | Dock |
| F | F | Ctrl+U | University |
| G | G | Ctrl+G | Mining Camp |
| Z | Z | Ctrl+Z | Lumber Camp |
| X | X | Ctrl+Y | Monastery |
| C | C | Ctrl+M | Market |
| V | V | Ctrl+V | Castle |
| B | B | Ctrl+B | Barracks |

#### Thumb Keys:

| Key | Tap | Hold |
|-----|-----|------|
| GRP (left) | Sticky Ctrl (assign group) | Momentary numbers layer |
| SPACE (mid) | Space (go to selected) | - |
| ./H (right) | `.` (idle villager, 150ms) | `H` (go to Town Center) |

### AOE2 Numbers Layer (Layer 5) - Hold GRP to access

```
|  F3  |  1  |  2  |  3  |  4  |  5  |
| DEL  |  6  |  7  |  8  |  9  |  0  |
| SHIFT| HOME|  ,  |  .  | ESC |  /  |
```

#### Control Group Workflow:

| Action | How |
|--------|-----|
| Select group 3 | Hold GRP + tap 3 |
| Assign group 3 | Tap GRP, then tap 3 (sticky Ctrl) |

## AoE2:DE Default Hotkeys

AoE2:DE uses a **grid layout** for selected-object commands, mapping actions to the left hand:

```
Top Row:    Q  W  E  R  T
Middle Row: A  S  D  F  G
Bottom Row: Z  X  C  V  B
```
